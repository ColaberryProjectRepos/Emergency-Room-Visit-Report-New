Step 8 - Patients' Visit by Month - Clustered Column Chart 17



  • Insert a Clustered Column Chart:
  • From the Visualizations pane, select the Clustered Column Chart.
  • Drag the Month field to the Axis area.
  • Drag the Total Visit field to the Values area.
  • Add Line Values:
    • Drag the following fields into the Secondary Values area:
      • Visit_WoW%: Color = #6B007B
      • Refer_WoW%: Color = #0E1A77
      • Sat_WoW%: Color = #AD5129
      • Ref_rate%: Color = #7E7912
      • Wait_time_WoW%: Color = #744EC2
      • Sat%: Color = #760404
    • Set Line Style:
      • Line Style: Solid
      • Line Width: 3px
  • Configure Dual Y-Axes:
    • Ensure that one Y-Axis is set for the Total Visit (numerical values) and the other for the percentages.
  • Title and Legend:
    • Set the chart title to "Patients' Visits by Month, Gender Breakdown."
    • Add the Gender field to the Legend area.
    • Ensure that the legend reflects all the line values with distinct colors (as specified above).
  • Formatting:
    • Assign distinct colors to each line value.
    • Set column colors based on gender:
      • Female: Color = #9AB8B6
      • Male: Color = #5F828D
      • NC (Not Classified): Color = #118DFF
