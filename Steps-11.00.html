Step 2 - Data Transformation 11
  • Open Power Query Editor: In Power BI, go to the Home tab and click on Transform data. This will open the Power Query Editor.
  • Add a Custom Column:
    • In the Power Query Editor, click on Add Column in the top menu and then select Custom Column.
  • Create the "Age Group" column:
    • In the dialog that appears, enter a name for your new column (e.g., Age Group).
    • Use the following logic for the custom column formula:

= if [patient_age] <= 2 then "Infancy"

  else if [patient_age] <= 6 then "Early Childhood"

  else if [patient_age] <= 12 then "Middle Childhood"

  else if [patient_age] <= 18 then "Teenager"

  else "Adults"

Create the "Age Range" column:

  • In the dialog that appears, enter a name for your new column (e.g., Age Range).
  • Use the following logic for the custom column formula:

= if [patient_age] <= 10 then "0-10"
  else if [patient_age] <= 20 then "11-20"
  else if [patient_age] <= 30 then "21-30"
  else if [patient_age] <= 40 then "31-40"
  else if [patient_age] <= 50 then "41-50"
  else if [patient_age] <= 60 then "51-60"
  else if [patient_age] <= 70 then "61-70"
  else "70+"

  • Create the "WaitTimeGroup" column:
    • Similarly, add a new custom column and use the following logic to categorize patient wait times:

= if [patient_waittime] <= 15 then "0-15"

  else if [patient_waittime] <= 30 then "16-30"

  else if [patient_waittime] <= 45 then "31-45"

  else if [patient_waittime] <= 55 then "46-55"

  else "55+"

  • Create the "patient_admin_binary" column:
    • Similarly, add a new custom column and use the following logic to replace TRUE with 1 and FALSE with 0:

= if [patient_admin_flag] = true then 1 else 0

  • Handle null values: Instead of removing null values, we recognize that they represent missing ratings. Replace these null values with 0 to maintain data integrity.
