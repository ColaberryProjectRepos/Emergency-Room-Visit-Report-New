Step 1 - Importing Data from Excel 10
  1. Open Power BI Desktop.
  2. Download dataset: Click the red button on the top of the page to download the dataset
  3. Import Data:

    • Click on Home in the ribbon.
    • Select Get Data 
    • Excel Workbook.
    • In the file dialog, navigate to and select the "Hospital_ER.csv" file
    • Click Open.
    • Click Transform Data to load the data into Power Query Editor.
    • In the Navigator window, select Hospital_ER
