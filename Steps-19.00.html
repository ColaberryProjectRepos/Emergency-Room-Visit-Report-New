  • To create the "Satisfaction Score by Age Range" chart, which is a combination of a Clustered Column and Line Chart, follow these steps:
    • Insert a Clustered Column and Line Chart:
      • From the Visualizations pane, select the Clustered Column and Line Chart.
    • Set Up the Axis and Values:
      • Drag the Age range field to the Shared Axis area.
      • Drag the Satisfaction Score field to the Column Values area.
      • Drag the Satisfaction Rate field to the Line Values area.
    • Configure Dual Y-Axes:
      • Ensure one Y-Axis shows the Satisfaction Score and the other shows the Satisfaction Rate percentage.
    • Title and Formatting:
      • Set the chart title to "Satisfaction Score by Age Range."
      • Column color: #5B7F7D
