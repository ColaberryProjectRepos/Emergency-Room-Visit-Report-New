Step 13 - Heatmap with a Dynamic Title Based on Parameter Selection 21



To create the Heatmap with a Dynamic Title Based on Parameter Selection, follow these steps:

  • Insert a Matrix Visualization:
    • From the Visualizations pane, select the Matrix visualization.
  • Set Up Rows and Columns:
    • Drag the Patient_Race field to the Rows area.
    • Drag the Age group field to the Columns area.
  • Create the Parameter (Toggle):
    • Create a Parameter using a slicer or buttons to switch between Avg. Satisfaction Score and Avg. Wait Time.
    • This parameter will be used to dynamically switch the values displayed in the matrix.
    • Detailed steps are illustrated in the accompanying GIFs.
  • Set Up Values:
    • Drag the Selected Metric measure you just created into the Values area of the matrix visualization.
  • Configure Conditional Formatting:
    • Apply conditional formatting to the Selected Metric values to show a color scale, where darker shades represent either higher satisfaction scores or lower wait times based on the selected parameter.
  • Create a Dynamic Title:
    • Create a dynamic title using a DAX measure that updates based on the selected parameter. The dynamic title will show different messages depending on whether the Avg. Satisfaction Score or Avg. Wait Time is selected.
    • Example DAX measure for the title:

Hitmap Caption =

    var _SelectedMeasure =

    SELECTEDVALUE(Parameter[Parameter Order])

    RETURN

    IF(_SelectedMeasure = 1,

    "The darker on the scale denotes LOW waiting time on the Age group",

    "Patient are most SATISFIED when the scale shows darker on the Age-group")

  • Apply Conditional Formatting to the Heatmap:
  • Apply the following color formatting to the matrix visualization based on the selected parameter:
    • For Average Satisfaction Score:
      • Lowest Value: #EAF4F4
      • Midpoint Value: #ACD3CA
      • Highest Value: #084A37
    • For Average Wait Time:
      • Lowest Value: #311E3D
      • Midpoint Value: #807F89
      • Highest Value: #D3DCE5
