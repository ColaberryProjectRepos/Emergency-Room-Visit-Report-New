Step 12 - Waterfall Chart 20


 

 

  • Insert a Waterfall Chart:
    • From the Visualizations pane, select Waterfall Chart.
  • Create the Breakdown Table:
    • Use the following DAX to create the Breakdown table:
      Breakdown = DATATABLE(
         "KPI", STRING,
         "KPISort", INTEGER,
         {
             {"Visit %", 1},
             {"Sat %", 2},
             {"Wait_time", 3},
             {"Referred %", 4}
         }
      )
  • Create the WoW Variance Measure:
    • Use the following DAX to create the WaterFall_measures measure:
      WaterFall_measures =
      VAR visit_wow = 'Calculations'[Visit WOW%]
      VAR refered_pat = 'Calculations'[Refer WOW%]
      VAR wait_time = 'Calculations'[Avg WaitingTime WOW%]
      VAR sat_score = 'Calculations'[Avg Satisfaction WOW%]
      RETURN
      SWITCH(
          SELECTEDVALUE(Breakdown[KPISort]),
          1, visit_wow,
          2, sat_score,
          3, wait_time,
          4, refered_pat
      )
  • Set Up Axis and Values:
    • Drag KPI (from the Breakdown table) to the Category axis.
    • Drag WaterFall_measures to the Y-Axis (Values) area.
  • Title and Formatting:
    • Set the chart title to "WoW Variance."
    • Adjust colors in the Format pane:
      • Increase: #529062
      • Decrease: #CA505C
      • Total: #466A6B
